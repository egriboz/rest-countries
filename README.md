# Country Information

Get information about countries via a RESTful API


## Development

```
$ git clone https://github.com/egriboz/rest-countries.git
$ yarn
$ yarn dev

or

$ npm install
$ npm run dev
```

This will serve the files at `http://localhost:3000/`
